3. Montaje físico del dispositivo robótico

Lista de materiales

Placa Arduino UNO (o compatible)
Módulo GPS 6MU2
Cables jumper macho-hembra (mínimo 4)
Protoboard (opcional, para facilitar conexiones)
Módulo adaptador de voltaje (si el módulo GPS no acepta directamente 5V)
Cable USB para conexión del Arduino al ordenador
Ordenador con software Arduino IDE instalado
Fuente de alimentación (puede ser desde el USB del ordenador o batería externa si se prueba de forma autónoma)

Instrucciones paso a paso

Adaptaciones DUA

🛠️ Adaptaciones DUA
Para garantizar la participación de todo el alumnado, este proyecto incorpora recursos adaptativos y accesibles que permiten ajustar el nivel de complejidad, los canales de acceso a la información y las formas de expresión.

✅ Guías paso a paso con pictogramas y ejemplos visuales.

✅ Subtítulos en vídeos.

✅ Instrucciones en formato digital con opción de ampliación.

✅ Instrucciones en formato de audio o explicaciones orales grabadas.

✅ Trabajo cooperativo con roles definidos para fomentar la participación activa.
